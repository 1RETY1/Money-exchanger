# Currency Exchange (Обменник валют)

Консольное приложение на C# для имитации обмена валют.

## Описание
Программа позволяет:
- Вводить начальный баланс в рублях и долларах
- Выбирать операцию обмена (рубли → доллары или доллары → рубли)
- Производить расчет с учетом заданных курсов валют
- Отображать итоговый баланс

## Курсы обмена (заданы в коде)
- 1 USD = 64 RUB (при покупке долларов)
- 1 USD = 66 RUB (при покупке рублей)
